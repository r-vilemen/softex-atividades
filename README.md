# Softex Atividades

Este projeto é uma atividade de componentes da Softex.

## Tecnologias Utilizadas

- **Next.js**
- **React**
- **Styled Components**
- **TypeScript**
- **HTML**

## Como Executar

1. Clone o repositório:
   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Execute o projeto:
   ```bash
   npm run dev
   ```
